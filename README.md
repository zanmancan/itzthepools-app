# Itz The Pools

Multi-league, multi-sport pools platform (Next.js + Supabase). App is a PWA and will target iOS/Android installs.

---

## Quick Start (Dev)

```bash
# 1) Install deps
npm i

# 2) Create a local env file
cp .env.example .env.local

# 3) Run the dev server on http://localhost:3001
npm run dev
```
